Manual install (all platforms)

Download the archive for your OS and architecture from GitHub Releases, extract it, and place the panex binary on your PATH.

Get started

After installing, verify it works:

panex version

Then create your first extension project:

panex init my-extension
cd my-extension
panex dev

Follow the on-screen instructions to load the extension in Chrome.

Load in Chrome

  1. Open chrome://extensions
  2. Turn on Developer Mode (top right)
  3. Click Load unpacked
  4. Select the .panex/dist folder printed by panex dev
