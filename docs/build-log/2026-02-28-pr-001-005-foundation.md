## PR1 - Repository Bootstrap

  - Initialized Go module contract in `go.mod:1-3`.
  - Added lint policy and pinned Go target in `.golangci.yml:5-33`.

## Reusable team lessons
1. Contract-first work (CLI -> config -> protocol -> transport) minimizes backtracking.
2. Every feature PR included negative-path tests, not only happy paths.
3. ADRs captured irreversible or high-impact choices near implementation time.
4. Citation-based PR narratives make reviews faster and onboarding cleaner.
