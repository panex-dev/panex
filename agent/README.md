# Panex Dev Agent

Chrome extension (Manifest V3) that connects to the Panex daemon and executes development commands such as `command.reload`.

## Local development

```bash
cd agent
pnpm install
pnpm run check
pnpm run test
pnpm run build
```

Load unpacked extension in Chrome using the `agent/` directory.
The manifest points to `dist/background.js`, generated by `pnpm run build`.
Protocol definitions are imported from the workspace package entrypoint `@panex/protocol`.

## Multi-extension targeting

For the default single-extension Panex config, the agent uses `extensionId = "default"`.

When Panex runs with `[[extensions]]`, each loaded dev-agent instance must store a matching `panex.extensionId` value in `chrome.storage.local` so targeted `command.reload` messages reach the correct extension only.
