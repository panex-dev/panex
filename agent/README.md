# Panex Dev Agent

Chrome extension (Manifest V3) that connects to the Panex daemon and executes development commands such as `command.reload`.

## Local development

```bash
cd agent
pnpm install
pnpm run check
pnpm run test
pnpm run build
```

Load unpacked extension in Chrome using the `agent/` directory.
The manifest points to `dist/background.js`, generated by `pnpm run build`.
